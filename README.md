# Sidelake Dynamic Wallpaper

## Installation

1. `git clone gitlab:tm/lakeside.git ~/Pictures/Wallpapers/Lakeside`
2. `cd ~/Pictures/Wallpapers/Lakeside`
3. `chmod +x install.sh update_paths.sh`
4. `./install.sh`
